## Gradient Noise Experiments

- 모든 실험은 **Google Colab 환경**에서 실행됨.
- 그래디언트에 **노이즈를 추가**한 결과, 사용자의 데이터 보안성이 일부 확보됨.
- 그러나 성능 하락 가능성이 존재하며, **최적 노이즈 수준**에 대한 추가 연구가 필요함 (예: trade-off 분석).

## Future Directions

- 단순히 노이즈를 추가하는 수준에서 벗어나,  
  **업데이트 손실(loss)을 최소화하면서 보안성을 확보할 수 있는 수학적/최적화 기반 알고리즘** 탐색 필요.
- 이 접근을 통해 **보안-성능 균형**을 보다 정교하게 설계 가능.

## Usage Manual

- 현재 폴더에 있는 `.ipynb` 파일을 **Google Colab**에서 실행하면 실험이 작동함.  
  - 단, 경로(path) 설정이 올바르게 되어 있어야 함.
- 실험 코드는 구글 드라이브 마운트로 경로를 설정했으며 단일 이미지와 다중 이미지(readme에는 파일 명으로 작성)실험을 위해서는 `auto.jpg`의 다운로드 및 적절한 경로 설정이 필요함.
